# outputs/

Tukaj agent piše svoje rezultate — povzetke, poročila, generirane
dokumente, dnevnike obdelave.

Tipične datoteke:
- `processing-log-YYYY-MM-DD.md` — kaj je agent premaknil pri obdelavi inboxa
- `insights-feedback-YYYY-MM-DD.md` — sinteza povratnih informacij strank (lekcija 5)
- `goodbrew-investor-update-YYYY-MM.pptx` — investorski deck (lekcija 6)
- `goodbrew-prodaja-YYYY-MM.xlsx` — mesečni Excel pregled (lekcija 7)
- `landing-draft.md` — osnutek pristajalne strani (lekcija 8)
- `my-A-to-E.md` — tvoj zapolnjen A–E zemljevid (lekcija 8)

Vsebina te mape je *tvoje delo* — kar agent generira, kar ti pošiljaš
naprej, kar gre v PowerPoint ali Excel preko datoteke ("file bridge").

Občasno počisti — ohrani pomembno, briši začasno.
