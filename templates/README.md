# Project Instruction Templates

Use these files when creating your own project folder:

- `CLAUDE.md` - Claude-specific project instructions
- `AGENTS.md` - portable agent instructions for other tools

Put them next to:

- `PERSON.md`
- `COMPANY.md`
- `BRAND.md`

The instruction files do not replace identity files. They tell the agent to read the identity files first and define the default working protocol, boundaries, and output format.
