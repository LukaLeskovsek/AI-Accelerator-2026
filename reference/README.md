# reference/

Datoteke, ki jih agent **bere, vendar ne spreminja**.

Sem postavi:
- pogodbe (ki jih ne smeš urejati)
- predloge (Word, PowerPoint, Excel templates)
- brand guidelines, ki jih je sprejela ekipa
- pravne dokumente
- karkoli, kar je samo za referenco

Razlika od `processed/`: tam so datoteke, ki jih agent lahko sortira,
preimenuje, premakne. Tu so datoteke, ki jih *bere*, ko piše osnutek
ali odgovori. Vsebina ostane nespremenjena.

V lekciji 6 (PowerPoint deck) agent uporabi `reference/` (če imaš tam
predlogo) za določitev tonalnosti in vizualnega stila.
